hexo+git+nginx+阿里云/github page搭建个人博客

Posted on 2018-09-26 | In 技术 |

前言

博主最近有网页设计的课程，想到做个人博客，于是有了这篇教程。
踩了很多坑，终于有了自己的博客★,°:.☆(￣▽￣)/$:.°★ 。
hexo是免费的博客框架，详解请移步官网https://hexo.io/zh-cn/

开始前的准备

安装git

Windows用户

从https://www.git-scm.com/download/下载git安装包，一路next。。。

Linux用户

基于RPM

yum install -y git


基于debian
apt-get install git

安装node.js

Windows用户

从https://nodejs.org/en/download/下载node.js安装包，一路next。。。

Linux用户

基于RPM

yum install -y nodejs

基于debian
apt-get install nodejs

Linux用户用包管理器安装的版本较旧，建议到官网下载源码安装

参考链接https://hexo.io/zh-cn/docs/

安装hexo

Windows用户

在任意盘新建文件夹，名字任取。
进入文件夹按住shift鼠标右键空白处，左键Git Bash Here

npm install hexo-cli -g  // 用npm包管理器安装hexo
hexo init blog  // 创建博客项目blog
cd blog  // 进入blog目录
npm install  // 安装依赖
hexo server  // 本地浏览

打开浏览器，网址栏输入localhost,回车即可看到hexo的欢迎页

Linux用户

cd /home
mkdir blog && cd blog
npm install hexo-cli -g  // 用npm包管理器安装hexo
hexo init blog  // 创建博客项目blog
cd blog  // 进入blog目录
npm install  // 安装依赖
hexo server  // 本地浏览

hexo使用简介

基本命令

hexo init  // 新建网站
hexo new  // 新建文章
hexo clean  //清除缓存
hexo g  //生成静态文件
hexo s  //启动服务器
hexo d  //部署网站

项目结构

.
├── _config.yml   // 站点配置文件
├── package.json  // 应用程序的信息
├── scaffolds // 模板文件夹
├── source // 用户资源
|   ├── _drafts
|   └── _posts
└── themes // 主题文件夹

详解请移步hexo文档https://hexo.io/zh-cn/docs/

把网站部署到GitHub page/云服务器

github

使用已有的github账号或者注册一个https://github.com/
。。。

进入个人主页，点击New repository新建一个仓库,仓库名一定要用你的账户名yourName.github.io。
然后修改站点配置文件_config.yml,在最后面添加

deploy:
  type: git
  repository: git@github.com:your_username/your_username.github.io.git  //注意后面的.git
  branch: master

保存退出，在git bash里
hexo g
hexo d

打开浏览器输入http://youName.github.io,回车查看

部署到云服务器（阿里云为例）

购买云服务器https://www.aliyun.com/?utm_content=se_1000301881

云服务器配置

安装contos 7.4

yum install -y git
wget http://nginx.org/download/nginx-1.15.4.tar.gz
tar -zxvf nginx-1.15.4.tar.gz
cd nginx-1.15.4
./configure
make && make install
adduser git
passwd git
cd /home/git
su git
mkdir blog  && cd blog
mkdir hexo
chown -R git:git /home/git/blog/hexo
chmod -R 755 /home/git/blog/hexo
git init --bare /home/git/blog/hexo.git
vi /home/git/blog/hexo.git/hooks/post-update.sample

全删，然后写入
#!/bin/bash
git --work-tree=/home/git/blog/hexo --git-dir=/home/git/blog/hexo.git checkout -f

:wq保存退出，然后
mv /home/git/blog/hexo.git/hooks/post-update.sample /home/git/blog/hexo.git/hooks/post-update
chmod +x /home/git/blog/hexo.git/hooks/post-update

最后配置nginx钩子vi /var/local/nginx/conf/nginx.conf,设置server部分
root /home/git/blog/hexo

本地配置

修改站点配置文件_config.yml

deploy:
  type: git
  repository: git@yourip:/home/git/blog/hexo.git
  branch: master
